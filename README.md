# gorent

Go Rent is an front-end for vehicle rent system built on React.js

## Installation

1. Clone the repository

   ```bash
   git clone https://github.com/chirzul/fe-gorentall.git
   ```

2. Install dependencies

   ```bash
   npm install
   ```

4. Run the app

   ```bash
   npm start
   ```

:fire: You're all set!
